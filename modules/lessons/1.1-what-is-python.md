## ğŸ“– What is Python?

Python is a **programming language**. Think of it as a way to give instructions to your computer. Just like you might give directions to a friend ("turn left, walk 100 meters, turn right"), you give instructions to a computer using Python code.

Here's the beautiful thing about Python: it was designed to be **easy to read and write**. The creator, Guido van Rossum, wanted a language that felt almost like writing English.

Let's look at an example:

```python
# Python code to check if a number is positive
number = 10

if number > 0:
    print("This number is positive")
```
Even if you've never programmed before, you can probably guess what this code does. That's the beauty of Python!
