## üöÄ Getting Started

### Step 1: Set Up Your Environment

**Option A: Local Setup (Recommended)**

```bash
# Install Python 3.10 or higher
# Download from python.org

# Create a virtual environment
python -m venv ds_env

# Activate it
# On Windows:
ds_env\Scripts\activate
# On Mac/Linux:
source ds_env/bin/activate

# Install required packages
pip install -r requirements.txt
```

Option B: Cloud Setup (No installation needed)

Use Google Colab:

1. Go to colab.research.google.com

2. Click File > Open Notebook

3. Select GitHub tab

4. Paste this repo URL

5. Open any .ipynb file

### Step 2: Clone This Repository
```bash
git clone https://github.com/yourusername/python-for-data-science-ai.git
cd python-for-data-science-ai
```
### Step 3: Start with Module 1, Lesson 1
Go to 01-Python-Fundamentals/01-Introduction-to-Python/lesson.md and begin!
